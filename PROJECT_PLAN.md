# Project Goal

The eCommerce Backend API is a RESTful API built using Node.js, Express.js, and MongoDB. It allows users to browse products, manage shopping carts, and enables administrators to manage products.

---

# User Stories

- As a shopper, I can view products.
- As a shopper, I can create a cart and add items to it.
- As an admin, I can add, update, and delete products.
